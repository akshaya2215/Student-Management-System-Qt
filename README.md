# 🎓 Student Management System (Qt)

A desktop-based **Student Management System** developed using **C++** and **Qt Widgets**. The application allows users to manage student records through Create, Read, Update, and Delete (CRUD) operations.

---

## ✨ Features

- 🔐 Login Authentication
- ➕ Add Student
- 🔍 Search Student
- ✏️ Update Student
- 🗑️ Delete Student
- 🚪 Logout

---

## 🛠️ Technologies Used

- C++
- Qt Widgets
- Qt Designer
- Git
- GitHub
